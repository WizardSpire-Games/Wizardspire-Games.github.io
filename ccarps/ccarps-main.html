CCARPS

Community Codex Adaptive RolePlay System - Live Beta

Welcome to CCARPS (Community Codex Adaptive RolePlay System). CCARPS is a skill based RPG system developed by a small team of family and friends. Our continuing mission to create a fun, fast-paced, RolePlaying Game is driven by our passion for creative fun. We are currently building out a live beta web-based, navigable, book that can be linked and bookmarked by topic. The RC3 version of the core rules document is available as a single page web document as is the RC2 World Framework (see link below) that can be used and adapted to your needs. We only ask that if you make changes that improve your game please share back to the community.
